# Translation Utilities for Brazilian Real Estate Data

This file contains functions for translating Portuguese column names and
values to English following geobr package patterns and real estate
domain conventions. Translate Dataset Column Names and Values

## Usage

``` r
translate_dataset(data, dataset_name)
```

## Arguments

- data:

  Dataset (tibble or list of tibbles)

- dataset_name:

  Name of the dataset for context-specific translations

## Value

Translated dataset with English column names and standardized values

## Details

Apply standard Portuguese to English translations for a given dataset.
Follows geobr package conventions and real estate domain standards.
