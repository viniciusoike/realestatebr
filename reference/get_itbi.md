# Get Property Tax Records

Imports and cleans property tax records for main Brazilian Cities.
Currently available only for Belo Horizonte and São Paulo. Unlike other
functions, there is no cached data available.

## Usage

``` r
get_itbi(city = "bhe")
```

## Arguments

- city:

  Must one of 'bhe', 'spo', 'Belo Horizonte', or 'São Paulo'

## Value

A `tibble`
