# Harmonize FipeZap Data for Stacking

Harmonize FipeZap Data for Stacking

## Usage

``` r
harmonize_fipezap_for_stacking(dat, transaction_type = NULL)
```

## Arguments

- dat:

  FipeZap data tibble

- transaction_type:

  "sale" or "rent" to filter for stacking

## Value

Harmonized tibble with standard RPPI columns
