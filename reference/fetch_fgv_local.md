# Read and process FGV IBRE data from a local CSV file

Read and process FGV IBRE data from a local CSV file

## Usage

``` r
fetch_fgv_local(path = "data-raw/xgdvConsulta.csv")
```

## Arguments

- path:

  Path to the `xgdvConsulta.csv` file downloaded from the FGV portal.

## Value

Tibble with FGV IBRE indicators in long format.
